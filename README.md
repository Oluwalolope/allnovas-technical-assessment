# All Novas Landing Page - Frontend Assessment

This project is a **complete landing page** built as part of the **technical assessment for a Frontend Developer position at All Novas**. The landing page demonstrates modern UI/UX design, responsive layout, and dynamic interactions using **React**, **TypeScript**, and **Tailwind CSS**.

---

## Features

* **Modern Landing Page Design**

  * Clean, professional layout suitable for showcasing a product or service.
  * Includes hero section, informational sections, call-to-actions, and footer.

* **Responsive & Mobile-First**

  * Fully responsive design that adapts to mobile, tablet, and desktop screens.
  * Ensures readability and usability across all devices.

* **Dynamic Components**

  * Smooth transitions, hover effects, and interactive elements.
  * Reusable components such as buttons, cards, and search bars.

* **Tailwind CSS Styling**

  * Utility-first styling for rapid development and consistent design.
  * Custom colors, spacing, typography, and shadows.

* **Clean Code & Type Safety**

  * Written in **TypeScript** for type safety and better maintainability.
  * Component-based structure for modularity and scalability.

---

## Technologies Used

* **React** (Functional Components & Hooks)
* **TypeScript**
* **Tailwind CSS**
* Optional: Extendable with animation libraries like Framer Motion.

---

## Usage

1. Clone the repository:

```bash
git clone https://github.com/Oluwalolope/allnovas-technical-assessment.git
```

2. Install dependencies:

```bash
npm install
# or
yarn install
```

3. Start the development server:

```bash
npm run dev
# or
yarn run dev
```

4. Open your browser at `http://localhost:5173` to view the landing page.
